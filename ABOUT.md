# 📋 Tentang Sistem Absensi QR Code

> Solusi modern untuk pencatatan kehadiran siswa yang cepat, akurat, dan efisien.

---

## 🎯 Visi Proyek

Sistem Absensi QR Code hadir untuk menggantikan metode absensi konvensional berbasis buku kertas dengan sistem digital yang:

- ✅ Meminimalkan kecurangan dalam pencatatan kehadiran
- ✅ Mempercepat durasi pengambilan absen
- ✅ Memudahkan guru dan staf administrasi dalam mengelola data kehadiran siswa

---

## ✨ Fitur Utama

| Fitur | Deskripsi |
|---|---|
| 📂 **Manajemen Data Siswa** | Import data siswa dari CSV/Excel (Nama, NISN, Kelas) |
| 🔲 **QR Code Unik** | Setiap siswa memiliki QR Code sebagai identitas digital |
| 📷 **Scanner Real-time** | Antarmuka web responsif terhubung kamera untuk pemindaian langsung |
| ⚡ **Pencatatan Instan** | Data kehadiran tersimpan otomatis dengan timestamp yang akurat |
| 📊 **Laporan Kehadiran** | Pantau kehadiran siswa secara langsung dan mudah |

---

## 🛠️ Teknologi yang Digunakan

```
Frontend  : HTML5, CSS3, Tailwind CSS
Logika    : JavaScript (DOM Manipulation & Attendance Logic)
Database  : Firebase Firestore (Cloud & Real-time)
Arsitektur: Single File Architecture
```

---

## ⚙️ Cara Kerja

```
1. 📤 Registrasi Data
   Administrator mengunggah data siswa melalui menu manajemen

        ↓

2. 🔲 Generate QR Code
   Sistem otomatis menghasilkan QR Code unik untuk setiap siswa

        ↓

3. 📷 Proses Absensi
   Siswa/petugas menunjukkan QR Code ke kamera pemindai

        ↓

4. ✅ Verifikasi & Simpan
   Sistem memvalidasi QR Code dan mencatat kehadiran ke database
```

---

## 🏫 Manfaat bagi Sekolah

- ⏱️ **Efisiensi** — Menghemat waktu guru dari pencatatan manual
- 🎯 **Akurasi** — Menghindari *human error* pada pencatatan nama atau NISN
- 🔍 **Transparansi** — Data kehadiran dapat dipantau langsung dan tersimpan aman di cloud
